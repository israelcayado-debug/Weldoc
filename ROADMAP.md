# Roadmap MVP por sprints

Plan de entrega del MVP para el sistema, con sprints de 2 semanas.

## Sprint 1: Fundaciones

- [x] Usuarios, roles y permisos.
- [~] Auditoria basica.
- [x] Catalogo de clientes.
- [x] Proyectos multi-proyecto (alta y configuracion minima).
- [x] Repositorio documental basico (subida y versionado).

## Sprint 2: WPS/PQR (ASME IX + ISO 15614-1)

- [~] Formularios guiados.
- [~] Validaciones esenciales por norma.
- [ ] Rangos de aprobacion en tiempo real.
- [x] Repositorio de WPS/PQR.
- [ ] Importacion basica.

## Sprint 3: WPQ/WQTR (ASME IX + ISO 9606-1/4)

- [~] Formularios guiados.
- [ ] Rangos de cualificacion en tiempo real.
- [~] Continuidad y alertas de vencimiento.
- [x] Repositorio de soldadores.
- [ ] Importacion basica.

## Sprint 4: Weld mapping, welding map y welding list

- [~] Welding Map por proyecto con marcado en planos y navegacion.
- [~] Welding List por proyecto con filtros y estados.
- [~] Carga de planos y marcado de soldaduras.
- [~] Creacion automatica de soldaduras.
- [ ] Trazabilidad minima: materiales, heat numbers y consumibles basicos.
- [ ] Inspeccion visual por etapas y estado.

## Sprint 5: Reportes, exportaciones y dossier final

- [~] Reportes minimos (avance y vencimientos).
- [ ] Exportaciones a Excel.
- [ ] Dossier final v1 en PDF con seleccion de contenidos.
